    bruce-singlePropAnimation mixin

    Uses the React.addons.CSSTransitionGroup syntax to create simple single property animations.

    @include bruce-singlePropAnimation($name: <string>, $duration: <string>, $prop: <string>, $fromEnter: <string>, $toEnter: <string>, $fromLeave: <string>, $toLeave: <string>);

    Example

    scss
    // scss
@include bruce-singlePropAnimation('fade', 300ms, opacity, 0, 1);

// css
.fade-enter {
    opacity: 0;   
}
.fade-enter.fade-enter-active {
    transition: all 300ms;
    opacity: 1;
}
.fade-leave {
    transition: all 300ms;
    opacity: 1;
}
.fade-leave.fade-leave-active {
    opacity: 0;
}

    bruce-breakpoint mixin

    Returns a media query based on the content of $bp.

    @include bruce-breakpoint($breakpoint: <string|value|list>);

    Example

    scss
    .class {
    font-size: 1rem;
    
    @include bruce-breakpoint('small') {
        font-size: 2rem;
    }

    @include bruce-breakpoint(40em) {
        font-size: 3rem;
    }

    @include bruce-breakpoint(max-width: 40em) {
        font-size: 4rem;
    }

}
