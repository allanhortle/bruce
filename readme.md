# Bruce

A utility base for Sass. Nothing Schmancy.

```
npm install --save-dev bruce
```


## Getting Started
bruce is a bunch of Sass helpers. To use it just import as you like, before your real sass. 

```
@import "bruce";

@import "myActuallySass";
@import "otherSass";
@import "moreSass";
@import "embarrassingSass";
```
